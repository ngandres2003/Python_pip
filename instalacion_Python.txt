Los paquetes que se están importando sirven para proporcionar herramientas y 
funcionalidades necesarias para compilar y desarrollar software en el sistema

–

build-essential ➡️ es un paquete que incluye herramientas de compilación y 
desarrollo de software necesarias para construir programas en C y C++

–

libssl-dev ➡️ es el desarrollo de la biblioteca OpenSSL, que proporciona funcionalidades 
de seguridad para aplicaciones de red, como cifrado, autenticación y verificación de certificados

–

ibffi-dev ➡️ es el desarrollo de la biblioteca libffi, que permite a las aplicaciones invocar
 funciones de otros programas en tiempo de ejecución

–

python3-dev ➡️ es el paquete de desarrollo de Python 3, que incluye los archivos de cabecera y otras 
herramientas necesarias para compilar módulos y aplicaciones de Python
